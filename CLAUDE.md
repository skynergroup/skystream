
# CLAUDE.md - Quick Start & Conventions

## Quick Start

1.  **Install dependencies:** `npm install`
2.  **Run development server:** `npm run dev`
3.  **Open application:** `http://localhost:3000`

## Key Conventions

*   **Components:** All components are located in `src/components`.
*   **Styling:** CSS Modules with BEM-like naming (`.component__element--modifier`).
*   **State Management:** Local component state with React Hooks.
*   **API:** Services for TMDB and streaming providers are in `src/services`.
*   **Testing:** Jest and React Testing Library for unit and integration tests.
